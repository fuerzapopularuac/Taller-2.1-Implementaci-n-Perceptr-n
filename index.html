Conclusiones por caso de prueba

📌 Fashion-MNIST

CNN con 2 capas convolucionales logró ~92% de accuracy. Limitación: imágenes externas con baja resolución generan confusión entre clases similares (zapatilla vs bolso).

🌿 CIFAR-10

Precisión 75.67% demostró necesidad de arquitecturas más profundas. El color y objetos no centrados exigen mayor abstracción. El dropout evitó sobreajuste moderado.

📸 SVHN

Modelo robusto a dígitos reales (88.7% test). Sin embargo, requiere segmentación previa para imágenes con múltiples números. El ruido y oclusiones afectan confianza.

🔤 EMNIST (tu aporte)

Alto número de clases (62) impacta rendimiento: confusiones O/0, I/l. Arquitectura de 3 capas conv + Dropout(0.3) esencial. Recomendación: más épocas y EMNIST Balanced.

Sugerencias transversales para futuras mejoras:
• Aumento de datos (rotación, zoom, desplazamiento) para mejorar generalización.
• Implementar aprendizaje por transferencia con redes preentrenadas (ResNet/EfficientNet) en Tiny ImageNet y datasets complejos.
• Ajuste de hiperparámetros (learning rate decay, early stopping).
• Para EMNIST: usar la variante Balanceada (47 clases) y aumentar épocas a 15-20.
